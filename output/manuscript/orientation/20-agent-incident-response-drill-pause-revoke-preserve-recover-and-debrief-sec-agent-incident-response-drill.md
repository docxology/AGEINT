## Agent incident response drill: pause, revoke, preserve, recover, and debrief {#sec:agent-incident-response-drill}

| Phase | Drill action | Artifact |
|---|---|---|
| Prepare | define agent scope, tool permissions, logs, escalation roles, and stop conditions | incident play card and contact matrix |
| Detect | identify anomalous output, policy drift, source-laundering, data exposure, or unsafe tool request | synthetic incident ticket and evidence snapshot |
| Contain | pause the workflow, revoke tool access, preserve records, and route to human review | containment decision and access-change log |
| Recover | restore a known-safe prompt, dataset, tool profile, or rubric state | rollback note and retest result |
| Debrief | record root cause, learner impact, rights impact, vendor implication, and refresh trigger | post-incident memo and owner assignment |
